给大家介绍一款，100%免费、静态、基于Github Pages、支持Markdown标记语言的 个人博客方案 —— Simple

首先，您需要一个Github账号，打开 http://github.com ，填入你的昵称、电子邮件、密码，点击Sign up for Github，在下个页面点击Finish sign up即可。

然后在Dashboard页面点击New repository，新建一个以你的用户名.github.io为名字的新repo，记得一定要选中Initialize this repository with a README。

最后，进入 http://editor.yangjinkun.cn页面，使用你的Github账户登录，点击Initialize按钮，稍等片刻，出现Initialzition has completed!后，你的博客就新建起来了。

点击下方的Go按钮，进入博客后台。点击左方的 New post 或者 New Page 来新建一篇博客或一个页面。

在右方的上面填写博客或页面的标题、URL（必须是 xxx.html，一定要加一个.html）、日期、标签（以空格区分多个标签）。在下方书写你的文字。写完后点击右上方的'save'按钮来保存本篇博客或页面。

那么，你的博客就正式建立好啦，博客的网址是： http://你的用户名.github.io

本文中新建的博客为： http://amtia.github.io

在你博客左边导航栏里点击 EDITOR 即可快速进入后台编辑。

Github Page 有10分钟的缓存时间，更新博客后需要10分钟左右的时间才能看到。

高级设置：

如果想绑定域名，你可以在repo中点击CNAME，在点击Edit，把你的域名添加进去保存即可。

然后把你的域名 Cname 到 你的用户名.github.io 或 A记录 到 你的用户名.github.io所在IP。
